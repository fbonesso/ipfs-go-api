# IPFS Go API
Go Lang API for InterPlanetary File System (IPFS).

## Usage
`go run main.go`

This will download the dependencies into $GOPATH.

## Methods
| Methos | Name | Description
|---|---|---|
|**`[POST]`**|/ipfs/add|Add file to IPFS.|
|**`[POST]`**|/ipfs/run/{command}|Run a command.|
